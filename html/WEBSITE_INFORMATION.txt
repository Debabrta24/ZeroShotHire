PROJECT OVERVIEW
================================================================================
ZeroShotHire is a comprehensive AI-powered career guidance and development 
platform designed to help individuals discover their ideal career path, develop
necessary skills, and achieve their professional goals.

MISSION
To empower individuals with personalized career guidance through AI-driven 
insights, helping them make informed decisions about their professional future.

USAGE INSTRUCTIONS
================================================================================

GETTING STARTED:
1. Open index.html in a web browser
2. Explore features from the landing page
3. Navigate using the sidebar menu
4. Toggle dark/light theme as preferred
5. Fill out career analysis to get recommendations

NAVIGATION:
- Use sidebar menu to access all pages
- Click logo to return to landing page
- Use browser back/forward buttons
- Bookmark favorite pages

DATA PERSISTENCE:
- Theme preference saved automatically
- Some data saved in browser storage
- Data resets when browser storage is cleared
- No server-side persistence (demo version)
